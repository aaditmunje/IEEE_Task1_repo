Q1) What is Machine Learning?
Machine Learning is a field of computer science where computers learn patterns from data and use those patterns to make predictions or decisions without being explicitly programmed. This is the definition, but in simple words some tasks, such as language proficiency, etc., where humans are better than computers. ML is giving the machine so much data that it can learn and predict data, making human tasks easier.

Q2) What is supervised learning?
Supervised learning is the learning wherein the input and output(target) are both provided while training the model. Here, machine learning is used for labeled data.

Q3) What is Unsupervised Learning?
This is the exact opposite of supervised learning, wherein the model is working with data that does not have labeled outputs. It analyzes the data and tries to find patterns or groups within the data.
Two common types of unsupervised learning are:
Clustering – The model groups similar data points together.
Dimensionality Reduction – The model reduces the number of variables in the data while keeping the important information.
